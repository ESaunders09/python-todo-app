# To-Do List CLI App

## Overview

A simple To-Do List app written in Python. This program lets users manage tasks directly from the terminal with features to **view, add, remove, and save tasks**. Tasks are saved to a file so they save between sessions.

## Features

- **View tasks** – Displays all current tasks in a numbered list.
- **Add tasks** – Add a new task with a description.
- **Remove tasks** – Remove a task by specifying its number.
- **Persistent storage** – Tasks are saved to a `tasks.txt` file and loaded automatically when the program starts.
- **Input validation** – Prevents invalid input for task numbers and menu options.
- Clean and readable **terminal output** with spacing for a better user experience.

## How to Run

1. Make sure you have **Python 3** installed.
2. Clone or download this repository.
3. Open a terminal in the project folder.
4. Run the program with:

```bash
python todo.py
```
